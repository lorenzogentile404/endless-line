# endless-line

A script to draw a line.

### Controls

- `a`: Down-left 
- `s`: Down
- `d`: Down-right

### Example

```console
(base) user@host:~$ python endless_line.py 
 \
  \
   \
    \
     \
     |
     |
      \
       \
        \
         \
         |
          \
           \
            \
             \
              \
              |
               \
              /
             /
            /
           /
           |
           |
          /
         /
        /
       /
       |
      /
     /
    /
   /
  /
```
